# IPv4 to Binary Converter

A minimalist, didactic web tool that converts IPv4 addresses to binary with a live step-by-step breakdown of the conversion process.

## Features

- **Live conversion** — results update as you type
- **Didactic breakdown** — each octet shows the subtraction method with powers of 2 (128, 64, 32, 16, 8, 4, 2, 1)
- **Zero dependencies** — no build step
